# chat_project
Мини-чат на Django с регистрацией, отправкой сообщений и архивом.

## Функции
- Регистрация пользователя через форму
- Авторизация хранится в `session` (если пользователь не вошёл — редирект на регистрацию)
- Чат:
  - отправка сообщений
  - лента сообщений (новые сверху)
  - фильтрация по дате через `?date=YYYY-MM-DD`
- Управление сообщениями (для автора):
  - редактирование своего сообщения
  - перемещение в архив (soft delete через `is_deleted`)
- Архив:
  - просмотр удалённых сообщений пользователя
  - восстановление сообщения из архива
- Уведомления (успех/ошибки) через `django.contrib.messages`

## Стек
- Python
- Django
- SQLite (по умолчанию)

## Установка и запуск
Виртуальное окружение:
python -m venv venv
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate
